# bookstore
C#언어와 asp, bootstrap, postgresql로 구현된 도서구매 사이트입니다.


주요 기능  
계정관리 - 가입, 탈퇴, 로그인, 회원정보수정, 메일인증  
도서관리 - 책 검색, 장바구니, 주문
